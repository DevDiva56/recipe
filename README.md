# 🍛 Grandma's Famous Pilau – Recipe Page

A simple HTML and CSS project built as part of my Full Stack Development learning journey.

This project focuses on building a beautiful recipe webpage while practicing the HTML and CSS concepts I have learned so far. Rather than creating a plain recipe, I wanted to tell a story around one of my favorite meals—Grandma's Famous Pilau.

---

## 📖 Project Overview

The webpage includes:

- A page title
- A hero section with an image
- A short story introducing the recipe
- Ingredients list
- Step-by-step cooking instructions
- A closing section sharing the meaning behind the recipe

The goal was to practice structuring webpages with semantic HTML while adding basic styling using CSS.
